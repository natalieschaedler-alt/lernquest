# LearnQuest 🎮

KI-Lernspiel für Schüler. Eigenen Lernstoff eingeben → magisches Abenteuer spielen.

## Setup

1. Clone repo
2. `npm install`
3. `.env.local` mit Keys ausfüllen (siehe `.env.example`)
4. Supabase: SQL aus `supabase/schema.sql` ausführen
5. `npm run dev`

## Tech Stack

React + TypeScript + Vite | Motion | Supabase | Anthropic API | Vercel
